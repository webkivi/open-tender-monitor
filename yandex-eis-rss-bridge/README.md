# EIS RSS Bridge for Yandex Browser

This unpacked Yandex Browser extension reads the official RSS feed generated by the EIS search page and sends matching entries to the local Open Tender Monitor. It does not bypass CAPTCHA, change IP addresses, or automate login.

## Installation

1. In Yandex Browser open `browser://extensions`.
2. Enable developer mode and choose **Load unpacked extension**.
3. Select this `yandex-eis-rss-bridge` folder.
4. Open the extension settings and enter the RSS link displayed by EIS, the local dashboard address, and the locally generated ingest token.

The bridge uses the official RSS feed at a configurable interval of at least 30 seconds. The dashboard remains the source of truth for deduplication, rules, notifications, and statuses.
